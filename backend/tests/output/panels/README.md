# Panel Outputs

Panel run outputs are written to this folder.

Each panel should use its own subfolder or clearly named output files.

Outputs may include:
- `analysis_result.json`
- `insight_graph.json`
- `replay_manifest.json`

Naming convention (lowercase with underscores):
- raw PDF: `ab_full_panel.pdf`
- parsed fixture: `ab_full_panel.json`
- output folder: `ab_full_panel/`
