**Backend** (`backend/core/models/results.py`):
```python

    analysis_id: str
    context: AnalysisContext
    biomarkers: List[BiomarkerScore]

    insights: List[BiomarkerInsight]
    overall_score: Optional[float]
    risk_assessment: Dict[str, Any]
    recommendations: List[str]

    processing_time_seconds: Optional[float]
```

**Frontend** (`app/types/analysis.ts`):
```typescript
interface AnalysisResult {
  analysis_id: string;
  result_version: string;
  biomarkers: BiomarkerResult[];
  clusters: ClusterData[];
  insights: InsightData[];
  recommendations: string[];
  overall_score: number | null;
  meta: {
    confidence_score?: number;
    processing_metadata?: Record<string, any>;
  };
  created_at: string;
}
```

**Differences**:
1. вљ пёЏ Backend has `context: AnalysisContext`, frontend has `meta` object
2. вљ пёЏ Backend has `risk_assessment: Dict[str, Any]`, frontend missing this field
3. вљ пёЏ Backend has `processing_time_seconds`, frontend missing this field
4. вњ… Both have `result_version` field for version compatibility
5. вњ… Core fields (biomarkers, clusters, insights, recommendations) match

**Recommendation**: Align frontend types with backend DTOs, or add explicit DTO transformation layer
