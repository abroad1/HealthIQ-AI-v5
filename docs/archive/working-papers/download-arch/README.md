# Download-Arch: Working Papers Archive

**Status:** ARCHIVE — working papers, drafts, and superseded documents.

This directory contains documents downloaded or produced during the design and planning phases of HealthIQ AI (approximately 2025 through early 2026). It includes:

- Early strategy drafts (v1.0 through v1.5 revision history — the adopted version is `docs/HealthIQ_AI_Strategic-Vision-and-12-Month-Sprint-Plan_v1.5_FINAL_ADOPTED.md`)
- Frontend design exploration papers and feedback files (v1–v5 results journey versions — the current version is `docs/strategy/HealthIQ_Final_Results_Journey_Recommendation_Paper_v6.md`)
- Intelligence model design workshop outputs
- Research pipeline proposals and early knowledge bus design thinking
- Superseded Automation Bus SOP versions (v1.1, v1.2 — active is `docs/AUTOMATION_BUS_SOP_v1.3.1.md`)
- GTM feedback rounds from multiple models
- UX design packages (v1–v4 — v5 is canonical in `docs/architecture/Frontend/`)
- Early questionnaire design documents
- Miscellaneous one-off investigation outputs

**None of these documents are authoritative for current product decisions.**

They are preserved because they contain reasoning and design history that may be useful for understanding how decisions were reached. They should not be cited as current truth.

For current truth, see:
- `docs/README.md` — live entry point
- `docs/AUTHORITY_MAP.md` — document status classification
